## C

- **[raylib](https://www.raylib.com)** - Uma biblioteca simples e fácil de usar para aproveitar a programação de videogames. [[código-fonte]](https://github.com/raysan5/raylib)
- **SGDK** - Um kit de desenvolvimento gratuito que permite o desenvolvimento de software em linguagem C para o Sega Mega Drive. [[código-fonte]](https://github.com/Stephane-D/SGDK)
- **[Simple DirectMedia Layer](https://github.com/libsdl-org)** - Uma biblioteca de desenvolvimento multiplataforma projetada para fornecer acesso de baixo nível a áudio, teclado, mouse, joystick e hardware gráfico. [[código-fonte]](https://github.com/libsdl-org/SDL)
- **Xash3D FWGS Engine** - Xash3D FWGS é uma engine de jogos que visa fornecer compatibilidade com a Engine do Half-Life e estendê-la, além de oferecer aos desenvolvedores de jogos um fluxo de trabalho bem conhecido. [[código-fonte]](https://github.com/FWGS/xash3d-fwgs)
- **[ioquake3](https://ioquake3.org)** - Versão open source da engine do Quake III Arena, otimizada para FPS e mods, com suporte a OpenGL e áudio avançado. Ideal para jogos de tiro em primeira pessoa. [[código-fonte]](https://github.com/ioquake/ioq3)
- **[id Tech 3](https://en.wikipedia.org/wiki/Id_Tech_3)** - Engine clássica da id Software para jogos 3D em tempo real, usada em títulos icônicos como Quake III; agora totalmente open source para extensões e ports. [[código-fonte]](https://github.com/id-Software/Quake-III-Arena)
